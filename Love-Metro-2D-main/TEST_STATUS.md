### Проблемные области

#### 1. ScoreCounterTests (ИСПРАВЛЕНО частично)
- **Проблема**: ScoreCounter требует компоненты TMP_Text, Animator
- **Решение**: Добавлены необходимые компоненты в Setup()
- **Статус**: Должны работать после рекомпиляции Unity

#### 2. PassengerRegistry / GameplayIntegrationTests
- **Проблема**: Singleton Instance с `private set` нельзя сбросить между тестами
- **Симптомы**: NullReferenceException в Setup/Teardown
- **Возможное решение**:
  - Использовать `[TestFixture]` с `OneTimeSetUp` вместо `SetUp`
  - ИЛИ создавать уникальные GameObject для каждого теста без сброса Instance
  - ИЛИ добавить публичный метод `ResetInstance()` для тестирования

#### 3. ManualPairingManager Tests
- **Проблема**: Camera.main и другие Unity-specific компоненты
- **Статус**: Частично исправлено (Camera tag установлен)
- **Требуется**: Дополнительная проверка

#### 4. PassengerAbilities Tests
- **Проблема**: Тесты способностей требуют ScoreCounter
- **Возможная проблема**: ScriptableObject инициализация
- **Требуется**: Проверить после перезапуска Unity
