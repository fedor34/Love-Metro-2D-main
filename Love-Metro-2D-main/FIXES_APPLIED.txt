ПРОБЛЕМЫ НА СКРИНШОТАХ:
───────────────────────
1. ❌ Розовые окна (missing textures)
   Причина: Слои фона отключены (m_IsActive: 0)
   
2. ❌ Missing script errors в Console
   Причина: Unity не скомпилировал скрипты после очистки кэша

ПРИМЕНЁННЫЕ ИСПРАВЛЕНИЯ:
────────────────────────
✓ Создан скрипт AutoEnableBackground.cs
  - Автоматически включает объект "Background"
  - Автоматически включает все дочерние слои (город_фон, город_дальний, и т.д.)
  - Запускается при старте каждой сцены

✓ Очищен кэш компиляции Unity
  - Удалён Library/ScriptAssemblies
  - Удалён Library/StateCache/ScriptAssemblies
  - Unity перекомпилирует все скрипты

ЧТО ДЕЛАТЬ СЕЙЧАС:
──────────────────
1. В Unity Editor: Window → General → Console → Clear (очистить консоль)

2. Подождите компиляции:
   - Внизу экрана Unity должен появиться индикатор "Compiling C# scripts..."
   - Дождитесь завершения (несколько секунд)

3. Запустите игру (Play)

4. Проверьте:
   ✓ Розовых окон больше НЕ должно быть
   ✓ В консоли должны появиться логи:
     [AutoEnableBackground] ✓ Background включен
     [AutoEnableBackground] ✓ Включен слой: 6_город_фон
     [AutoEnableBackground] ✓ Включен слой: 5_город_дальний
     ...
   ✓ Ошибки "missing script" должны исчезнуть

ЕСЛИ ОШИБКИ ОСТАЛИСЬ:
──────────────────────
1. Закройте Unity полностью
2. В PowerShell выполните:
   Remove-Item -Recurse -Force "Love-Metro-2D-main\Library"
3. Откройте Unity снова (полная пересборка, займёт 1-2 минуты)
