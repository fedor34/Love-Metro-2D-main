# 🚨 КРИТИЧЕСКИЕ ИНСТРУКЦИИ - ПРИНУДИТЕЛЬНАЯ ПЕРЕСБОРКА

## Проблема:
Unity может не пересобрать скрипты после моих изменений из-за кэша.

## ✅ ЧТО СДЕЛАТЬ (2 варианта):

### Вариант 1 (БЫСТРЫЙ):
1. **Закройте Unity Editor** (если открыт)
2. **Удалите папку** `Love-Metro-2D-main/Library/ScriptAssemblies/`
3. **Откройте Unity Editor** снова
4. Подождите пока проект пересоберется (может занять 1-2 минуты)

### Вариант 2 (из Unity Editor):
1. **Assets → Reimport All**
2. Подождите завершения
3. **Edit → Project Settings → Player → Other Settings**
4. Измените **Scripting Define Symbols** (добавьте любой символ, например `FORCE_REBUILD`, затем удалите)
5. Unity автоматически пересоберет скрипты

---

## ⚠️ Про ошибку UISprite.psd:

Ошибка `Failed to find UI/Skin/UISprite.psd` - это **НЕ критично**.
- Это встроенный Unity UI Skin который отсутствует
- Игра запускается несмотря на это
- Не влияет на мои исправления

Но если хотите убрать:
1. **Edit → Project Settings → UI**  
2. Установите **Default Sprite** на любой существующий спрайт

---

## 🔍 КАК ПРОВЕРИТЬ ЧТО СКРИПТЫ ПЕРЕСОБРАЛИСЬ:

### После открытия Unity Editor:

1. **Откройте Console** (Window → General → Console)
2. **Нажмите Clear** (правый верхний угол)
3. **Запустите игру** (Play)
4. **Ищите в Console мои логи:**

```
[Passenger][vip-collider] PassangerMale_VIP(Clone): ORIGINAL offset=(0.00, 0.17) size=(1.69, 0.56)
[Passenger][vip-collider] PassangerMale_VIP(Clone): Collider already configured correctly, skipping normalization
```

Если эти логи **ЕСТЬ** → скрипты пересобрались! ✅

Если этих логов **НЕТ** → скрипты НЕ пересобрались, попробуйте другой вариант ❌

---

## 🎮 ПОСЛЕ ПЕРЕСБОРКИ:

1. Создайте VIP персонажей в игре
2. Посмотрите логи от диагностических скриптов:
   - `[FindAllBoundaries]` - найдены ли границы?
   - `[CollisionDebugger]` - происходят ли коллизии?
3. Нажмите **F9** и **F10** для дополнительной диагностики
4. **Скопируйте все логи** и отправьте мне

---

## 💡 Дополнительно - если всё равно не работает:

### Полная очистка кэша:
1. **Закройте Unity Editor**
2. **Удалите папки:**
   - `Library/`
   - `Temp/`
   - `obj/`
3. **Откройте Unity Editor**
4. Подождите **полной пересборки** (может занять 5-10 минут)
