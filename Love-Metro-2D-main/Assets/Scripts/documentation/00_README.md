# Love Metro 2D - Техническая документация

## Обзор проекта

**Love Metro 2D** - это 2D игра-симулятор знакомств в метро, разработанная на Unity. Игрок управляет движением поезда, создавая инерцию, которая толкает пассажиров друг к другу. Цель игры - соединить пассажиров противоположного пола в пары.

## Быстрый старт

### Основные компоненты сцены

Для работы игры в сцене должны присутствовать:

1. **TrainManager** - управление поездом и инерцией
2. **PassangerSpawner** - спавн пассажиров
3. **PassangersContainer** - контейнер для хранения ссылок на пассажиров
4. **PassengerRegistry** - оптимизированный реестр пассажиров (создаётся автоматически)
5. **CouplesManager** - управление парами
6. **ScoreCounter** - подсчёт очков
7. **ClickDirectionManager** - обработка ввода игрока

### Основной игровой цикл

```
[Игрок нажимает/тянет мышь]
         ↓
[ClickDirectionManager определяет направление]
         ↓
[TrainManager создаёт инерцию]
         ↓
[Пассажиры получают импульс через startInertia делегат]
         ↓
[Пассажиры переходят в состояние Falling/Flying]
         ↓
[При столкновении M+F создаётся пара]
         ↓
[ScoreCounter начисляет очки]
```
